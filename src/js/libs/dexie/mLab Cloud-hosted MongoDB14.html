Conditional Operators

 Unique

    

        A unique flag of true signifies a unique index. The _id_ index is always unique.
    

    

        Parentheses indicate an implicit value. Documents representing index definitions do not explicitly include unique: false, so the value of unique is "(false)."
    





    
 Sparse

    

        A sparse flag of true signifies a sparse index, which
        only contains documents with the indexed field.  When sorting or
        filtering on a sparsely indexed field, any documents that are
        missing that field will be omitted from the query results.
    

    

        Note that a sparse index can only have one field.
    

    

        Parentheses indicate an implicit value. Documents representing index definitions do not explicitly include sparse: false, so the value of sparse is "(false)."
    





    
 Min / Max

    

        If min and max are blank, the index assumes lattitude/longitude and is thus configured for a -180 to 180 value range.
